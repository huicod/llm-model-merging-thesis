# Excluded From Public Release

This public repository was prepared from a larger local workspace. The following file groups were intentionally excluded.

## External Baseline Code

- `RankMean-main/`
- `project_editor/RankMean/`
- `server_code_data/RankMean/`

Reason: these directories contain upstream or copied baseline code. The public repository documents how the baseline was used but does not redistribute it.

## Model Artifacts

- Base model directories.
- LoRA checkpoints.
- Exported LoRA models.
- Merged model directories.
- Tokenizer export folders.
- `*.safetensors`, `*.bin`, `*.pt`, `*.pth`, `*.ckpt`.

Reason: these files are large and may be restricted by model licenses.

## Data and Results

- Original datasets.
- Processed dataset files such as JSON, JSONL, CSV, or Parquet outputs.
- Raw lm-eval result JSON files.
- TensorBoard event files.
- Generated figures and analysis images.

Reason: datasets and benchmark outputs should be regenerated by users from licensed sources and are not needed for a portfolio repository.

## Temporary Files

- `__pycache__/`
- `.ipynb_checkpoints/`
- notebook checkpoints.
- local logs.
- local machine paths and AutoDL scratch directories.

Reason: these are machine-specific and not useful for reproduction.
